About

A Django web application that displays sensor data on a map, allows comparison with regional estimates and can generate time-series graphs for each sensor location.

All data is loaded into the web app model/DB by the (admin) user at runtime, from 4 CSV files comprising 2 data files and 2 metadata files. See the github documentation section on how to load data for more detailed instructions.

Users can also upload their own data locally (with a browser/session), again via a CSV file upload, so that their own data can be compared alongside the actuals/estimates data pre-loaded.

The estimation methods (click on 'Select estimation method' and see all options except 'pre-loaded') use the estimation methods (classes) available in the python library: https://pypi.org/project/region-estimators ('concentric-regions' and 'distance-simple' at time of writing)
