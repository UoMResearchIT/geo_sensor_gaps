Upload data and visualise
Upload geographical time-series data as a CSV file. The csv file must have the following columns:

Column 1: has header 'timestamp' and contains timestamps as strings, matching those available on the timestamp slider,
Example: 2016-10-12

Column 2: has header 'geom' and contains geographical locations (currently points only),
Example: POINT (-2.254 53.397)

Columns remaining: header(s) will be used as measurement name(s) and values should be numeric (non numeric will be converted to null)

Note that the uploaded file's data is stored locally on the browser, and only used for visualisation on the map, within this browser session.

Data sources:

Site measurement data uses imputed data from MEDMI and AURN, described in: 'https://zenodo.org/record/4416028 (2016-2019 data) (file 3: imputed data) and in 'https://zenodo.org/record/4740965 (2020 data)


The region estimation methods (under 'Select estimation method'):
the 'pre-loaded' data contains estimated data downloadable from https://zenodo.org/record/4475652 (file 3: imputed data) (2016-2019 data) and in https://zenodo.org/record/5113200 (file 1: original data) (2020 data)

About:

A tool for estimating regional measurements based on known values and visualising the results.

The estimation methods (see 'Select estimation method': all options except 'pre-loaded') are those estimation methods (classes) available in the python library: https://pypi.org/project/region-estimators/ .

('concentric-regions and distance-simple at time of writing)
Initial instructions:

Click on a site to see site info and also the option to view site and estimated data across all timestamps.

Click on a region to see region info

If no sites exist for this timestamp, either:

(i) use slider to find another timestamp

(ii) use 'Select measurement' option to change measurements.
